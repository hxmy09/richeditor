### 目标
- 移除移动端不太适用的编辑功能
- 保证性能的稳定性
- 高度可定制的下底栏
- 风格统一
<br/>

### 支持版本

android4.4及以上版本

### 已有效果
- 粗体
- 斜体
- 删除线
- 引用块
- H1~H4字体
- 插入图片
- 插入链接
- 插入分割线
- 链接修改
- 图片点击删除
- 图片进度条现实
- 图片加载失败显示
- 字数显示
- 撤销（undo）
- 重做（redo）
- 发布内容
- 主题切换
<br/>
