### 目标
- 移除移动端不太适用的编辑功能
- 保证性能的稳定性
- 高度可定制的下底栏
- 风格统一
<br/>

### 即将完成
- 主题风格切换
- andriod4.0 ~ andriod4.4 js适配
- 图片在被软键盘删除时的回调
<br/>

### 已有效果
- 粗体
- 斜体
- 删除线
- 引用块
- H1~H4字体
- 插入图片
- 插入链接
- 插入分割线
- 点击已插入的链接回调
- 点击已插入图片回调
- 设置图片进度条
- 设置图片加载失败
- 字数监听回调
- 撤销（undo）
- 重做（redo）
- 异步（同步）获取HTML内容
- 标题栏聚焦（失去焦点）回调
- 光标所在位置状态回调
